You may running the java application typing this way

e.g

java -jar GoEuroTest.jar Berlin

In config.ini file there are parameters of configurarion.

HTTPS_JSON_GOEURO 
    -> Configures address that is located the API
CERT_DIG 
    -> Digital certificated
ARQ_CSV_PATH 
    -> Define the csv file path
APPEND_CSV_FILE 
    -> You can decide if you prefer to append the file or overwride it when you start the application.
